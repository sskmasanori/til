# Syntax Cheet Sheat

## Get element
```
$('***')
```

## Get value in element
```
$('***').text();
```

## Get value in input
```
$('***').val()
```

## Get data attribute value
```
$('***').data('data_attribute_name');
```

## Get location object properties
```
$(location).attr('href');
$(location).attr('search');
```
