## When Use Semicolon In JavaScript
2021.02.01
```
}で終わる場合は、基本的に;は省略できる。
```
```
if (true) console.log(true);
```
```
if (true) {
    console.log(true);
}
```
```
function learn() {
}
```
```
// 変数を宣言する文の一部であるため、;はいる。
const read = function() {
};
```
