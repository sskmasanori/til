## Create An Array Containing A Range Of Elements
2021.04.19

PHP:4,5,7,8

```
// [1,2,3,4,5]
range(1, 5);
```
```
// [0,10,20,30,50]
range(0, 50, 10);
```
```
// ['a', 'b', 'c', 'd']
range('a', 'd');
```
```
// ['a', 'c', 'e']
range('a', 'e', 2);
```
```
// ['c', 'b', 'a']
range('c', 'a');
```

[PHP公式: range](https://www.php.net/manual/ja/function.range.php)
