## how to make table
2021.03.19

```
col1 | col2 | col3
----- ------ -----
val1 | val2 | val3
val4 | val5 | val6
```
```
<table>
    <tr>
        <th>col1</th>
        <th>col2</th>
        <th>col3</th>
    </tr>
    <tr>
        <td>val1</td>
        <td>val2</td>
        <td>val3</td>
    </tr>
    <tr>
        <td>val4</td>
        <td>val5</td>
        <td>val6</td>
    </tr>
</table>
```
```
tr: table row
th: table header
td: table data
```
